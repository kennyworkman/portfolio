Gene alignment application

A minimal application for gene alignment built from scratch.
software biology

Try it. Source code.

I built a UI wrapper around common command line utilities for multiple sequence alignment using the linux package Clustal Omega .

Backend manipulation of clustal processes was done with python. An API was exposed using Flask . Application logic was containerized with Docker.

I used NGINX to build out a reverse-proxy server on a blank machine running an Ubuntu distribution. Gunicorn was used to communicate with NGINX and serve up the Dockerized apps exposed on a defined port.

More information on infrastructure and all source code is on GitHub.
